About Me

Lionel Andrés Messi Cuccittini is an Argentine professional footballer who plays as a forward and captains both Spanish club Barcelona and the Argentina national team.
The best player in the world and regarded by many as one of the greatest players of all time.
Messi has won a record-tying five Ballon d'Or awards, four of which he won consecutively, and a record five European Golden Shoes.
He has spent his entire professional career with Barcelona, where he has won a club-record 32 trophies, including nine La Liga titles, four UEFA Champions League titles and six Copas del Rey.
Both a prolific goalscorer and a creative playmaker, Messi holds the records for most goals scored in La Liga (408), a La Liga season (50), a club football season in Europe (73), a calendar year (91),
El Clásico (26), most hat-tricks in the UEFA Champions League (8), as well as those for most assists in La Liga (165) and the Copa América (11). He has scored over 670 senior career goals for club and country.

Diffrences between Web 1.0, Web 2.0 and Web 3.0:
Web 1.0 Web 2.0 Web 3.0
Mostly Read-Only Widely Read-Write Portable and Personal
Company Focus Community Focus Individual Focus
Home Pages Blogs / Wikis Live-streams / Waves
Directions Tagging User Behaviour
Web Forms Web Applications Smart Applications

Use Cases :
Web 1.0 Use Case ("readable" form):
A user "asks the web" for a query and probably would find non accurate results
like a static website with explanations about his query. the website would probably be
with basic design (frames and tables used to position and align elements).
Example: A user asks for "side effects of drugs" and finds a website with read only information about his query.
Web 2.0 Use Case ("writeable" form):
A user "communicates" with the web. Uses networking sites, shares his knowledge and opinions,
can share other content like music and videos. Other people can consume the content shared by the user.
Example: A user asks for "side effects of drugs" and finds information shared by others on social networks and sites like Wikipedia.
Web 3.0 Use Case ("executable" form):
A user interacts with a dynamic web, interactive services and machine to machine application.
The websites use AI algorithms to address user's needs (self-profiling), discovering new trends.
Better indexed knowledge and wide searching abilities (high level filtering). A user can create DB with these index abilities.
Most user communication with the machine goes to the "cloud" and managed there.
Example: A user asks for "side effects of drugs" and gets information from the search engine after being proccessed in AI algorithms so the data will fit the specific user.
